## Instructions

These steps will guide you through contributing to this project:

- Fork the repo
- Clone it and install dependencies

  ```bash
  git clone https://github.com/routexjs/routex-hooks
  cd routex-hooks

  npm install
  # or
  yarn
  ```

Make and commit your changes. Make sure the commands `npm run build`/`yarn build` and `npm run test`/`yarn test` are working.

Finally send a [GitHub Pull Request](https://github.com/routexjs/routex-hooks/compare?expand=1) with a clear list of what you've done (read more [about pull requests](https://help.github.com/articles/about-pull-requests/)).
